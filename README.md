## 보완사항

<br>

- 레트로 게임적인 이미지가 덜 구현된 상태입니다. <br>추가적인 분위기 구현이 필요합니다.
- Chart.js 각 그래프 항목의 font 들을 제대로 처치가 불가능합니다. 가능하다면, light / dark 모드 구현이 가능합니다.
